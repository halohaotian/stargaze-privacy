# Security Policy

## Reporting a vulnerability

If you believe you have found a security issue in **StarGaze: Night Sky** iOS app or in the contents of this privacy-policy repository, please report it privately rather than opening a public issue.

**Contact:** [halohaotianwu@gmail.com](mailto:halohaotianwu@gmail.com)

Please include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- The app version and iOS/iPadOS version
- Any proof-of-concept or screenshots

We will acknowledge receipt within 5 business days and aim to issue a fix or mitigation within 30 days for verified issues.

## Scope

This policy covers:

- The StarGaze iOS / iPadOS application
- The privacy-policy page published at <https://halohaotian.github.io/stargaze-privacy/>
- This GitHub repository (`halohaotian/stargaze-privacy`)

## Out of scope

- Issues in third-party services (e.g., the public ISS-tracking API at `api.wheretheiss.at`) — please report those to their respective operators.
- Bugs that do not expose user data or compromise app integrity. StarGaze does not collect personal data, so the realistic impact surface is narrow.

## Data practices

StarGaze does not collect, transmit (with the sole exception of the documented ISS API GET request), or store personal data. See [`index.html`](./index.html) for the full privacy policy.
